constituyen su proyecto elf_template.


elf_template.vcproj
    Éste es el archivo de proyecto principal para los proyectos de VC++
    generados mediante un Asistente para aplicaciones. 
    Contiene información acerca de la versión de Visual C++ con la que 
    se generó el archivo, así como información acerca de las plataformas,
    configuraciones y características del proyecto seleccionadas en el
    Asistente para aplicaciones.

Este proyecto le permite generar, limpiar y volver a generar desde 
Visual Studio llamando a los comandos insertados en el asistente. El 
comando de generación puede ser nmake o cualquier otra herramienta que 
utilice.

Este proyecto no contiene ningún archivo, por lo que no se muestra 
ninguno en el Explorador de soluciones.
